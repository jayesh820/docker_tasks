Run Graphical Applications from Docker

Learn how to display and interact with GUI software running inside Docker containers

Introduction

While Docker is primarily used for headless, server-style workloads, you can run full graphical applications inside containers. By configuring X11 forwarding or using VNC/noVNC, you can launch and interact with GUI-based software securely and efficiently.

Option 1: Using X11 Forwarding (Linux)

xhost +local:docker

# Run container with X11 socket mounted
docker run -it --rm \
  -e DISPLAY=$DISPLAY \
  -v /tmp/.X11-unix:/tmp/.X11-unix \
  my-gui-image

This method lets the container connect to your host's display server. Itâ€™s lightweight but only works locally and needs proper xhost permissions.

Pro Tip: Use xhost +SI:localuser:username instead of xhost + to grant access more securely.

Option 2: Using VNC or noVNC

With VNC, the GUI runs in a virtual desktop inside the container, accessible via a VNC client or browser.

docker run -p 5901:5901 -p 6080:6080 my-vnc-image
  • VNC Client: Connect to localhost:5901
  • noVNC: Open http://localhost:6080 in a browser

Option 3: Xpra for Persistent Sessions

Xpra allows seamless, persistent GUI sessions with better performance than VNC for certain apps.

Example: Running Firefox in Docker

docker run -it --rm \
  -e DISPLAY=$DISPLAY \
  -v /tmp/.X11-unix:/tmp/.X11-unix \
  jess/firefox

With VNC, you could use an image like dorowu/ubuntu-desktop-lxde-vnc and install Firefox inside.

Security Note: Avoid exposing VNC ports directly to the internet without authentication and encryption.

Best Practices

  • Limit container permissions (avoid --privileged unless necessary).
  • Use non-root users in containers when running GUI apps.
  • Leverage Docker volumes for persistent configuration and profiles.
  • Restrict network exposure for VNC/noVNC services.

Conclusion

Running GUI apps in Docker is a powerful way to sandbox complex software, test across environments, and streamline development workflows. Whether you choose X11, VNC, or Xpra, you can enjoy full desktop experiences within containers with the right setup.
